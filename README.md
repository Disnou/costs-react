**Note: this is a one-way operation. Once you `eject`, you can't go back!**

If you aren't satisfied with the build tool and configuration choices, you can `eject` at any time. This command will remove the single build dependency from your project.

Instead, it will copy all the configuration files and the transitive dependencies (webpack, Babel, ESLint, etc) right into your project so you have full control over them. All of the commands except `eject` will still work, but they will point to the copied scripts so you can tweak them. At this point you're on your own.

You don't have to ever use `eject`. The curated feature set is suitable for small and middle deployments, and you shouldn't feel obligated to use this feature. However we understand that this tool wouldn't be useful if you couldn't customize it when you are ready for it.

# Costs - Project Manager / Gerenciador de Projetos 💰

*( 🇺🇸 English | 🇧🇷 [Português](#-versão-em-português) )*

This is a cost and project management system developed in **React**. The application allows users to create projects, set budgets, and add services, automatically tracking expenses to ensure the established limit is not exceeded.

## 🚀 Technologies Used
* React.js
* React Router DOM
* CSS Modules
* JSON Server (Fake API)
* UUID (Unique ID generation)

## 🧠 What I Learned
During the development of this project, I was able to apply fundamental concepts of the React ecosystem in practice:
- Componentization and code reusability.
- State management and lifecycle with Hooks (`useState`, `useEffect`).
- Single Page Application (SPA) navigation using `react-router-dom`.
- Consuming and manipulating data from an API (GET, POST, PATCH, DELETE methods).

## 👏 Credits & Acknowledgments
This project was developed by following the React course by instructor **Matheus Battisti** from the [Hora de Codar](https://www.youtube.com/c/HoradeCodar) channel. All credits for the layout, business rules, and course didactics go to him. The material was essential for my hands-on front-end learning!
